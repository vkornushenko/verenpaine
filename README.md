# VerenPaine App
## Save and track your blood pressure readings.

### About
VerenPaine App is a simple study project built to practice and demonstrate skills in TypeScript, Next.js, Express, MongoDB, and modern full-stack web development.

### Fully deployed
Even though it's a study project, the application is fully deployed and available online:
[https://verenpaine.vercel.app/](https://verenpaine.vercel.app/)
- Backend deployed to Railway/Render
- Frontend to Vercel
- Database is running on MongoDB
